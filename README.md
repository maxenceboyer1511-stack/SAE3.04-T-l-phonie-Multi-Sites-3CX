## 📞 Description – SAE3.04 Téléphonie & Certification 3CX

La SAE3.04 consiste à découvrir, installer et maîtriser une solution de téléphonie IP moderne à travers la plateforme **3CX**, un IPBX largement utilisé en entreprise.  
L’objectif principal est de se former en autonomie via l’académie 3CX, puis de passer la **certification officielle 3CX Basic**, qui valide les compétences essentielles pour déployer un système de téléphonie VoIP.  
Le projet inclut également la réalisation d’une démonstration vidéo montrant l’installation, la configuration et le fonctionnement du système.

Cette SAE permet de comprendre les bases d’un IPBX, la configuration réseau associée (SIP, trunks, pare-feu), le provisioning des téléphones, ainsi que les bonnes pratiques de déploiement multi‑site.

---

### 🧩 Contenu de la formation 3CX

| Module | Thème | Objectif |
|--------|--------|-----------|
| 1 | Installer 3CX | Pré‑requis, FQDN, certificats, ports, installation Linux |
| 2 | Pare‑feu | Ouverture des ports, NAT, tests de connectivité |
| 3 | Applications 3CX | Softphones, provisioning, PUSH |
| 4 | Téléphones de bureau | Configuration, provisioning automatique |
| 5 | Trunks SIP | Connexion au fournisseur VoIP |
| 6 | Messagerie vocale | Configuration et gestion |
| 7 | Répondeurs numériques (SVI) | Menus vocaux interactifs |
| 8 | Groupes d’appels | Files, transferts, règles de routage |

Ces 8 modules constituent la base nécessaire pour réussir la certification **3CX Basic**.

---

### 🎓 Certification 3CX Basic

- 30 questions à choix multiple  
- 60 minutes pour terminer  
- 25 bonnes réponses minimum  
- 1 seule tentative (nouvel essai après 30 jours)  
- Score utilisé pour la notation de la SAE  

Avant de passer le test, il est recommandé d’ouvrir tous les chapitres dans des onglets séparés pour gagner du temps lors de la consultation.

---

### 🛠️ Travail demandé dans la SAE

- S’inscrire à l’académie 3CX  
- Suivre les 8 modules de formation (vidéos + supports PDF)  
- Prendre des notes pour faciliter la révision  
- Passer la certification Basic  
- Réaliser une **vidéo démonstrative** montrant :  
  - l’installation de 3CX  
  - la configuration de base  
  - le fonctionnement du système  
- Préparer un **PowerPoint** intégrant :  
  - extraits vidéo  
  - explications techniques  
  - étapes clés de l’installation  

---

### 🎯 Objectif final

L’objectif de la SAE3.04 est de maîtriser les bases d’un système de téléphonie IP moderne, de comprendre son installation, sa configuration réseau et son fonctionnement, puis de valider ces compétences via la certification officielle 3CX.  
Cette SAE prépare aux environnements professionnels où la VoIP et les IPBX sont omniprésents.
